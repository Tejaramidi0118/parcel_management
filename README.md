# Courier Management System

A comprehensive courier management platform with separate frontend and backend architecture.

## Project Structure

```
app-7j70tfy2ddkx/
├── frontend/          # React frontend application
├── backend/           # Backend API (to be implemented)
└── README.md          # This file
```

## Frontend

The frontend is a React + TypeScript application located in the `frontend/` directory.

### Getting Started

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npx vite --host 127.0.0.1
   ```

4. Open your browser at: `http://127.0.0.1:5173`

See `frontend/README.md` for more details.

## Backend

The backend folder is reserved for future backend implementation. Currently, the frontend uses mock data.

See `backend/README.md` for planned backend structure.
