## Quick Start Guide

1. **Visit the Home Page** to see the service overview
2. **Try the Fare Calculator** to estimate shipping costs
3. **Login as Customer** to track parcels and view dashboard
4. **Login as Courier** to manage deliveries and update status
5. **Login as Admin** to access all management features
